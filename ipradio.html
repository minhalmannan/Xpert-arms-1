IP NETWORK RADIO SYSTEM

This innovative product generates a new world of possibilities with radios that can leverage already existing Wi-Fi networks. Consider the possibility of having radio coverage anywhere you currently have coverage for Wi-Fi based devices. The controller that connects to the wireless network allows grouped users to hear and talk with another group. Individual calls are also possible. Full duplex communication (as with a regular telephone) can be used if enabled in the controller.

WLAN-based radio system offers scalable, license-free communication that uses standardized wireless network products as infrastructure. In several cases, said infrastructure is already in place.
